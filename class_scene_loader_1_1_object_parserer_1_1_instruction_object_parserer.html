Detailed Description

this class is responsible for send GET requests for Instruction Json files, and copy them to an scriptable object so, we can use it in other scripts. The reason why we have so many Json files is that Unity cannot read easily more than one level of nesting. It needs to be called on its separate scene with a loading bar. When we are done, we have an scriptable object that is filled with instruction text and our main scene can use it. User language for instructed text has been considered into account by loading proper Jsons from Remote or local.
