Detailed Description

this class is responsible for two step task implementation. It is designed in a way that it can be used just by change of input for both trial and real case scenario. The gameObject this script is attached to will be enabled when we are in task two. The picture assignment needs to do only one time trial and real case, and after that, it is just 3070 Probability and button swap, for each round.
