Detailed Description

This class is responsible for making a random swap for each round on DelayDiscounting. It uses an intermediary Sc to carry the result to the horizontal layout group. There is a value applier on horizontal layout group that will get updated when we change delayDiscountingButtonHorizontalSpacingSc value.
